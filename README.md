# My first app on React
## Consist of 3 main pages
* Sign In/ Sign Up page
* Map page itself
* About page
  * ...and logout modal

## Map Features
* Zoom In / Zoom Out
* Toggle markers on/off
* Load markers from data you saved
* Save markers to your account
* Select type of places you want to display on map
